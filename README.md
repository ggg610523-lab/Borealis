# Borealis (experimental portingkit for android)
<h1>
<center>
Wine Porting toolkit for Android</center>
</h1>
This is an experimental porting kit inspired by Winlator by brunoDev the main purpose behind this is to port an existing Windows game into android ecosystem.it is basically a wrapper for wine.
